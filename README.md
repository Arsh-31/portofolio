# Portfolio

A modern, responsive portfolio built with Next.js and TypeScript.

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React, React Icons
- **Deployment**: Vercel
